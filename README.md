# Gerenciador de Tarefas em Python

Sistema simples de gerenciamento de tarefas desenvolvido em Python, com menu interativo no terminal.

##  Funcionalidades
- Cadastrar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Exibir status da tarefa

##  Conceitos utilizados
- Programação Orientada a Objetos
- Listas
- Funções
- Laço while
- Estruturas condicionais
- Organização de código

##  Como executar
```bash
python main.py
